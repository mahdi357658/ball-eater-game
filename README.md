# Ball Eater Game

مجموعه دو بازی ساده با پایتون و pygame.

## نصب پیش‌نیاز

```
pip install pygame-ce
```

اگه با خطای نصب pygame مواجه شدی (مخصوصاً روی پایتون ۳.۱۴)، به‌جای `pygame` از `pygame-ce` استفاده کن — کاملاً جایگزین هست و بدون تغییر کد کار می‌کند.

---

## 🎮 بازی ۱: Ball Eater (`ball_game.py`)

بازیکن یک مار است که سرش دنبال موس حرکت می‌کند. باید توپ‌ها را بخورد و از بمب‌ها دوری کند.

**اجرا:**
```
python ball_game.py
```

**کنترل:**
- حرکت موس = حرکت بازیکن (مار)
- کلیک یا Enter = شروع / ادامه در صفحه‌ی شروع و پایان
- Esc = خروج از بازی

**قوانین:**
| نوع توپ | امتیاز | اثر روی بازیکن |
|---|---|---|
| ⚪ توپ ساده | +۲ | کمی بزرگ‌تر می‌شود |
| 🟡 توپ طلایی | +۱۰ | بزرگ‌تر می‌شود |
| 🔴 بمب | -۳۰ | کوچک می‌شود |

بازی ۲۰ مرحله دارد؛ هر مرحله باید به امتیاز هدف مشخصی برسی تا وارد مرحله بعد شوی. هر مرحله نسبت به قبلی سخت‌تر می‌شود (توپ‌ها سریع‌تر، تعدادشان بیشتر، بمب‌ها بیشتر).

## 🏎️ بازی ۲: Neon Highway - Car Racing (`car_racing.py`)

یک بازی رانندگی بی‌پایان (Endless Racer). باید با ماشینت از برخورد با ماشین‌های دیگر در جاده فرار کنی.

**اجرا:**
```
python car_racing.py
```

**کنترل:**
- کلیدهای چپ/راست (یا A / D) = حرکت ماشین به چپ و راست
- بعد از باخت: R = شروع دوباره، Q = خروج

**قوانین:**
- امتیاز به مرور زمان (فریم به فریم) افزایش می‌یابد.
- هر ۵۰۰ امتیاز یک سطح بالاتر می‌روی؛ سرعت ماشین‌های حریف بیشتر و فاصله ورودشان کمتر می‌شود.
- سه نوع ماشین حریف وجود دارد: ماشین اسپرت قرمز (سریع)، کامیون آبی (کندتر)، سدان زرد (استاندارد).
- برخورد با هر ماشین حریف = پایان بازی.

## 🛠️ رفع خطاهای رایج (Troubleshooting)

### خطای ۱: فایل پیدا نمی‌شود

```
python.exe: can't open file 'C:\Users\...\car_racing.py': [Errno 2] No such file or directory
```

**پاسخ:** یعنی مسیر فایل اشتباه است یا فایل هنوز دانلود نشده. اول با دستور `dir` مطمئن شو فایل توی همون پوشه‌ای هست که داری ازش اجرا می‌کنی، یا مسیر کامل و درست فایل رو بده. مثال:
```
cd Downloads
python car_racing.py
```

### خطای ۲: pygame نصب نیست

```
ModuleNotFoundError: No module named 'pygame'
```

**پاسخ:** باید pygame (یا pygame-ce) رو نصب کنی:
```
pip install pygame-ce
```

### خطای ۳: نصب pygame روی پایتون ۳.۱۴ شکست می‌خورد

```
error: subprocess-exited-with-error
Getting requirements to build wheel did not run successfully.
...
ModuleNotFoundError: No module named 'distutils.msvccompiler'
```

**پاسخ:** پکیج اصلی `pygame` هنوز نسخه‌ی آماده (wheel) برای پایتون ۳.۱۴ ندارد و pip سعی می‌کند از سورس کامپایلش کند که روی ویندوز شکست می‌خورد. به‌جایش نسخه‌ی `pygame-ce` رو نصب کن که از پایتون ۳.۱۰ تا ۳.۱۴ ساپورت می‌کند و کد را بدون تغییر اجرا می‌کند:
```
pip uninstall pygame
pip install pygame-ce
```
اگر باز هم مشکل داشت، مستقیم روی همون نسخه‌ی پایتون نصب کن:
```
py -3.14 -m pip install pygame-ce
```

### خطای ۴: پنجره‌ی بازی باز نمی‌شود یا بلافاصله بسته می‌شود

**پاسخ:** معمولاً به‌خاطر همون ارور نصب‌نشدن pygame است — دستورات بالا رو اجرا کن. اگر بازم مشکل بود، متن کامل خطایی که تو ترمینال میاد رو بفرست.
