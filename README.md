# Ball Eater Game

مجموعه دو بازی ساده با پایتون و pygame.

## نصب پیش‌نیاز

```
pip install pygame-ce
```

اگه با خطای نصب pygame مواجه شدی (مخصوصاً روی پایتون ۳.۱۴)، به‌جای `pygame` از `pygame-ce` استفاده کن — کاملاً جایگزین هست و بدون تغییر کد کار می‌کند.

## 🎮 بازی ۱: Ball Eater (`ball_game.py`)

بازیکن یک مار است که سرش دنبال موس حرکت می‌کند. باید توپ‌ها را بخورد و از بمب‌ها دوری کند.

**اجرا:**
```
python ball_game.py
```

**کنترل:**
- حرکت موس = حرکت بازیکن (مار)
- کلیک یا Enter = شروع / ادامه در صفحه‌ی شروع و پایان
- Esc = خروج از بازی

**قوانین:**
| نوع توپ | امتیاز | اثر روی بازیکن |
|---|---|---|
| ⚪ توپ ساده | +۲ | کمی بزرگ‌تر می‌شود |
| 🟡 توپ طلایی | +۱۰ | بزرگ‌تر می‌شود |
| 🔴 بمب | -۳۰ | کوچک می‌شود |

بازی ۲۰ مرحله دارد؛ هر مرحله باید به امتیاز هدف مشخصی برسی تا وارد مرحله بعد شوی. هر مرحله نسبت به قبلی سخت‌تر می‌شود (توپ‌ها سریع‌تر، تعدادشان بیشتر، بمب‌ها بیشتر).

## 🏎️ بازی ۲: Neon Highway - Car Racing (`car_racing.py`)

یک بازی رانندگی بی‌پایان (Endless Racer). باید با ماشینت از برخورد با ماشین‌های دیگر در جاده فرار کنی.

**اجرا:**
```
python car_racing.py
```

**کنترل:**
- کلیدهای چپ/راست (یا A / D) = حرکت ماشین به چپ و راست
- بعد از باخت: R = شروع دوباره، Q = خروج

**قوانین:**
- امتیاز به مرور زمان (فریم به فریم) افزایش می‌یابد.
- هر ۵۰۰ امتیاز یک سطح بالاتر می‌روی؛ سرعت ماشین‌های حریف بیشتر و فاصله ورودشان کمتر می‌شود.
- سه نوع ماشین حریف وجود دارد: ماشین اسپرت قرمز (سریع)، کامیون آبی (کندتر)، سدان زرد (استاندارد).
- برخورد با هر ماشین حریف = پایان بازی.
